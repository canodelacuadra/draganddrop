Drag an drop

tutorial

Arrastrar y soltar puede ser una manera muy intuitiva para que los usuarios interactúen con nuestro sitio o aplicación web. Las personas suelen utilizar arrastrar y soltar para cosas como:

  • Mover archivos dentro de carpetas
  • Cambios de orden de las listas de preferencias
  • Objetos en todo tipos en los juegos , como las cartas de una baraja o las piezas de un puzzle
